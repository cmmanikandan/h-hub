## 1. Profit Rules Flow

```
                    SELLER SETS PRICE
                          │
                          ↓
              ┌───────────────────────┐
              │   Find Matching Rule  │
              └───────────────────────┘
                          │
        ┌─────────────────┼─────────────────┐
        ↓                 ↓                 ↓
   ┌────────┐       ┌────────┐       ┌────────┐
   │ Rule 1 │       │ Rule 2 │       │ Rule 3 │
   │ ₹0-500 │       │₹501-2K │       │ ₹2K+   │
   │  25%   │       │  20%   │       │  15%   │
   └────────┘       └────────┘       └────────┘
                          │
                          ↓
              ┌───────────────────────┐
              │ Calculate % Profit    │
              │ Profit = Price × %    │
              └───────────────────────┘
                          │
                          ↓
              ┌───────────────────────┐
              │ Apply Min/Max Limits  │
              │ Floor: ₹50            │
              │ Cap: ₹500             │
              └───────────────────────┘
                          │
                          ↓
                  ADMIN PROFIT AMOUNT
```

## 2. SuperCoin Rules Flow

```
                    ORDER COMPLETED
                          │
                          ↓
              ┌───────────────────────┐
              │   Calculate Order     │
              │   Total Amount        │
              └───────────────────────┘
                          │
        ┌─────────────────┼─────────────────┐
        ↓                 ↓                 ↓
   ┌────────┐       ┌────────┐       ┌────────┐
   │ Rule 1 │       │ Rule 2 │       │ Rule 3 │
   │₹0-499  │       │₹500-2K │       │ ₹2K+   │
   │  1%    │       │  2%    │       │  3%    │
   └────────┘       └────────┘       └────────┘
                          │
                          ↓
              ┌───────────────────────┐
              │ Calculate Order Coins │
              │ Coins = Amount × %    │
              └───────────────────────┘
                          │
                          ↓
              ┌───────────────────────┐
              │ Add Rounding Coins    │
              │ Final - Raw Price     │
              └───────────────────────┘
                          │
                          ↓
                  TOTAL SUPERCOINS
                          │
                          ↓
              ┌───────────────────────┐
              │ Credit to Customer    │
              │ After Delivery        │
              └───────────────────────┘
```

## 4. Complete Pricing Pipeline

```
┌────────────────────────────────────────────────────────────────┐
│ STEP 1: INPUT                                                  │
├────────────────────────────────────────────────────────────────┤
│ Seller Price: ₹1000                                            │
│ Distance: 10 km                                                │
│ Payment: COD                                                   │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 2: APPLY LOGISTICS COSTS                                  │
├────────────────────────────────────────────────────────────────┤
│ ₹1000 + ₹30 (packing) + ₹50 (shipping) = ₹1080                │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 3: APPLY PROFIT RULE                                      │
├────────────────────────────────────────────────────────────────┤
│ Rule: ₹501-₹2000 → 20%                                         │
│ Profit: ₹1000 × 20% = ₹200                                     │
│ Ads: ₹70                                                       │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 4: CALCULATE GST                                          │
├────────────────────────────────────────────────────────────────┤
│ Base: ₹1080 + ₹200 + ₹70 = ₹1350                              │
│ GST: ₹1350 × 18% = ₹243                                        │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 5: ADD DELIVERY                                           │
├────────────────────────────────────────────────────────────────┤
│ Delivery: ₹20 + (10km × ₹2.5) = ₹45                           │
│ Fuel: 10km × ₹5 = ₹50                                         │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 6: CALCULATE RAW PRICE                                    │
├────────────────────────────────────────────────────────────────┤
│ ₹1080 + ₹200 + ₹70 + ₹243 + ₹45 = ₹1638                       │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 7: APPLY ROUNDING                                         │
├────────────────────────────────────────────────────────────────┤
│ Strategy: Nearest ₹10                                          │
│ ₹1638 → ₹1640                                                  │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 8: CALCULATE SUPERCOINS                                   │
├────────────────────────────────────────────────────────────────┤
│ Rule: ₹500-₹2000 → 2%                                          │
│ Order Coins: ₹1640 × 2% = 33 coins                             │
│ Rounding Coins: ₹1640 - ₹1638 = 2 coins                       │
│ Total: 35 coins                                                │
└────────────────────────────────────────────────────────────────┘
                            ↓
┌────────────────────────────────────────────────────────────────┐
│ STEP 9: SETTLEMENT DISTRIBUTION                                │
├────────────────────────────────────────────────────────────────┤
│ Seller:   ₹1080 (₹1000 + ₹30 + ₹50)                           │
│ Delivery: ₹-5   (₹45 - ₹50)                                   │
│ Admin:    ₹565  (₹200 + ₹70 + ₹243 + ₹50 + ₹2)               │
│ ─────────────────────────────────────────────────────────      │
│ Total:    ₹1640 ✓                                              │
└────────────────────────────────────────────────────────────────┘
```
